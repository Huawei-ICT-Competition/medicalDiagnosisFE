Overview

The accurate classification of Bone X-ray Images is crucial for the diagnosis and treatment of bone diseases. In this project, we present a Convolutions Neural Network (CNN) approach for Bone X-ray Image classification using TensorFlow. Our goal is to develop a model that can accurately classify bone images into their respective categories using deep learning techniques. We use image augmentation techniques to increase the variability of the input data and improve the model's performance.



Model Architecture

Our CNN model has three convolutional layers with max pooling, one dense layer, and a softmax output layer. We used ReLU activation function in the convolutional layers and the dense layer. We compiled the model with the categorical cross-entropy loss function and optimized with the Adam optimizer. To prevent overfitting, we added a dropout layer with a rate of 0.5 after the last convolutional layer.



Results

The model achieved an accuracy of 0.6763 and a loss of 0.5840 on the training dataset, while achieving an accuracy of 0.8050 and a loss of 0.4805 on the validation dataset. These results suggest that the model has better performance on the validation dataset and is not overfitting to the training dataset. We presented the results using confusion matrices to visualize the model's performance on each category.


*The model that achieved the mentioned results was trained and exported using ModelArts on the Huawei cloud.
